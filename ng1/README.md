# NG-1: JavaScript Basics & DOM

This module focuses on JavaScript fundamentals and basic DOM manipulation.

## What I learned
- Writing JavaScript in the script tag (placed at the end of body)
- Declaring variables and concatenating strings
- Creating and using functions
- Understanding the DOM and selecting elements using `getElementById`
- Taking user input and displaying dynamic output
- URL encoding for sharing links
