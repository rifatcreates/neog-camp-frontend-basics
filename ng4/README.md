# NG-4: Styling, DOM Queries & Theme Switching

This module focuses on building a real-world single page portfolio using HTML, CSS, JavaScript, and Pico CSS.

## What I Learned

- Building a single-page portfolio website
- Using Pico CSS for layout, cards, and styling
- Creating cards using Pico CSS components
- Implementing a theme switcher (light / dark)
- Using `querySelector` for DOM selection
- Using `setAttribute` to dynamically update HTML attributes
- Detecting system theme using `window.matchMedia`
- Understanding how theme preference works in browsers
- Structuring UI sections cleanly in a one-page layout

## Micro Apps / Projects

### Lessons (Instructor Guided)

- **Portfolio Page**
  - Single page personal portfolio
  - Multiple sections (about, skills, projects, etc.)
  - Theme switcher using Pico CSS
  - Responsive card-based layout
