# Cinego App

A movie watchlist Single Page Application built as part of NG-7 assignment.

## Pages

- Home  
  Displays movie list with basic information and view action.

- Details  
  Shows detailed movie information with remove option.

- Add Cinema  
  Allows adding new movies to the watchlist.

## Concepts Used

- API-driven SPA architecture
- Reusable routing structure
- Frontend-only integration with provided backend
- Add and delete operations using API
- Consistent SPA navigation
