# NG-7: Advanced SPA & API-based Routing

This module focuses on building multi-page Single Page Applications
using API-driven routing and data operations.

## What I Learned

- Handling SPA routing with multiple pages
- Working with pre-written backend APIs (frontend-focused learning)
- Fetching list and details data using API calls
- Adding new entries using API (POST)
- Removing entries using API (DELETE)
- Understanding data flow in SPA
- Using string.split for processing input data
