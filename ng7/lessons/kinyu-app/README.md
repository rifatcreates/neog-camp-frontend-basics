# Kinyu App

A lending management Single Page Application built during NG-7 lessons.

## Pages

- Home  
  Displays lending records in a table with:
  - Who
  - Type
  - Short details
  - View action

- Details  
  Shows detailed lending information including:
  - Name
  - Type
  - Short details
  - Lending date
  - Remove option

- Log Loans  
  A form page to add new lending entries.

## Concepts Used

- API-based SPA routing
- Pre-written backend integration
- Fetching list and detail data
- Deleting entries using API
- Adding entries using API
- String manipulation using split
