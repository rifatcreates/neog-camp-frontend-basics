# Multi Page Finance Apps

A multi-page web application built while learning HTML navigation and JavaScript logic.

## Included Apps

### Lesson Apps
- **Paise Planner**
  - Calculates monthly savings based on goal, duration, and current savings

- **Currency Converter**
  - Converts USD amount to INR

### Assignment Apps
- **Discount Deducer**
  - Calculates final price after discount
  - Shows amount saved

- **Bill Batwara**
  - Splits total bill among number of people

## Concepts Used

- Multi-page navigation
- HTML forms and input fields
- JavaScript calculations
- DOM manipulation
- Shared CSS styling
