# BT-1: Bootstrap Basics

This module focuses on learning Bootstrap fundamentals and building responsive multi-page websites using Bootstrap utility classes and components.

## What I Learned
- Using Bootstrap via CDN
- Bootstrap utility classes (bg, display, spacing, images)
- Responsive images using img-fluid
- Page layout using Bootstrap spacing helpers
- Lists and text formatting with Bootstrap
