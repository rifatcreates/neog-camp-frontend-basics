# NG-5: Arrays, Objects & Database Driven Apps

This module focuses on working with arrays, objects, templating,
and building database-driven applications using JavaScript.

## What I Learned

- Working with arrays and array methods like map
- Using objects to structure data
- Rendering UI using templating
- Understanding client-side databases
- Performing CRUD operations using TaffyDB
- Sorting and filtering data
- Building multi-page applications with shared database logic
- Handling DOMContentLoaded events
