# VibeBoard

A multi-page GIF management web application built during NG-5 lessons.

## Pages

- Home  
  Shows GIF count per category and recent GIFs

- Category  
  Displays GIFs by selected category

- Rating  
  Shows GIFs sorted by highest rating

- Likes  
  Shows GIFs sorted by most likes (descending)

- Add GIF  
  Adds new GIF data to the database

## Concepts Used

- Arrays and objects
- map and sorting techniques
- Client-side database (TaffyDB)
- CRUD operations
- DOMContentLoaded
- Templating
