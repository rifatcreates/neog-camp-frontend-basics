# BookShelf

A database-driven book management app built as part of NG-5 assignments.

## Pages

- Dashboard  
  Overview of books

- Category  
  Displays books by category

- Rating  
  Shows highest rated books first

- Likes  
  Shows most liked books first

- Add Book  
  Adds new book data to the database

## Concepts Used

- Arrays and objects
- Client-side database (TaffyDB)
- CRUD operations
- Sorting and filtering
- Multi-page navigation
