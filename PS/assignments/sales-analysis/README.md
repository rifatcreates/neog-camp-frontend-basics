# Sales Analysis

A two-page data analysis application built as part of PS assignment.

## Pages

- Sales Analysis (Home)
  - Key sales insights
  - Calculated and derived information
  - Data displayed using articles and tables

- All Sales Data
  - Complete sales information displayed in table format

## Concepts Used

- JavaScript for loops
- Iterating over arrays of objects
- Calculating derived data using helper functions
- Rendering data dynamically in UI
- Multi-page application structure
