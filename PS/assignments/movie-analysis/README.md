# Movie Analysis

A two-page data analysis application built as part of PS assignment.

## Pages

- Movie Analysis (Home)
  - Movie-related insights and statistics
  - Calculated and derived information
  - Data displayed using articles and tables

- All Movie Data
  - Complete movie information displayed in table format

## Concepts Used

- JavaScript for loops
- Working with arrays of objects
- Creating helper functions for extra data logic
- Dynamic UI rendering
- Multi-page application structure
