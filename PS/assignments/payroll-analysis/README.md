# Payroll Analysis

A two-page data analysis application built as part of PS assignment.

## Pages

- Payroll Analysis (Home)
  - Employee payroll insights
  - Calculated and derived information
  - Data displayed using articles and tables

- All Employee Data
  - Complete employee information displayed in table format

## Concepts Used

- JavaScript for loops
- Iterating through employee data
- Writing helper functions for calculations
- Rendering tables and articles dynamically
- Multi-page application structure
