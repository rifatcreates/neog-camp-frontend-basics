# Class Analysis

A two-page data-driven application built during PS lessons.

## Pages

- Class Analysis (Home)
  - Class topper details
  - A-grade students list
  - Class statistics
  - Data displayed using article and table formats

- All Students Data
  - Complete student information displayed in table format

## Concepts Used

- for loop for data rendering
- Array of objects
- Helper functions for grade calculation
- Dynamic UI updates using JavaScript
