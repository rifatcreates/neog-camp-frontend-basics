# Health Tools

A set of self-built health-related micro apps created as part of NG-3 assignments.

## Included Apps

- Calorie Compass  
  Calculates daily calorie needs based on age, gender, and activity level.

- Water Watcher  
  Suggests daily water intake based on body weight and season.

- Step Sankhya  
  Estimates daily steps needed to reach a target weight.

## Concepts Used

- Numeric calculations
- Input validation
- Conditional logic
- Error handling on UI
