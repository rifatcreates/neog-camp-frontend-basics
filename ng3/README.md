# NG-3: JavaScript Logic & Multi-Page Applications

This module focuses on applying JavaScript logic, handling user input validation,
and building real-world multi-page applications.

## What I Learned

- Converting string input to numbers using parseInt and parseFloat
- Formatting numeric output using toFixed
- Validating user input and showing errors on UI
- Building multi-page applications with shared navigation
- Writing logic-heavy JavaScript applications
- Structuring real-world mini projects
