# Petto App

A Single Page Application built during NG-6 lessons.

## Pages

- Home  
  Displays a list of pets with name, type, and a details link.

- Details  
  Shows detailed information of a selected pet.

## Concepts Used

- Single Page Application architecture
- Client-side routing using Atom Router
- JavaScript modules (import / export)
- array.find for fetching specific pet data
- Dynamic view rendering
