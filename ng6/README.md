# NG-6: Single Page Applications & Routing

This module focuses on building Single Page Applications (SPA),
understanding routing, and structuring JavaScript applications using modules.

## What I Learned

- Difference between multi-page and single-page applications
- How SPA routing works
- Configuring routes using Atom Router
- Defining routes and handling navigation
- Using JavaScript modules with import and export
- Finding specific data using array.find
- Rendering dynamic views without page reload
